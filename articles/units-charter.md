# Units Charter

## Unit registration

A top-level folder may become a unit when registered in `units/registry.md`.

## Unit records

Each unit record should define:

- purpose
- default risk level
- current owner status
- release authority expectations
- notes on public meaning of the unit

## Representation

Each active unit is entitled to one assembly seat.
