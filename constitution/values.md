# Values

## Core values

1. Legibility over hidden context
2. Constitutional order over convenience
3. Contestability over opaque authority
4. Rebuildability over team permanence
5. Public reason over unexplained coercion
6. Cross-functional coherence over silo optimization
7. Safe execution over reckless speed

## Value use

When a law, article, rule, or procedure is ambiguous, the interpretation that best preserves these values should be preferred unless a higher-order text clearly commands otherwise.
