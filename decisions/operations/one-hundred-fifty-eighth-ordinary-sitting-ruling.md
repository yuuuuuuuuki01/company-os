# One-Hundred-Fifty-Eighth Ordinary Sitting Ruling

## 裁定

- 裁定: `高リスク審査着手順議案` を採択し、最初の high-risk screening opening を `credential-ledger`、policy-first 別レーンを `saigai` として進める
- 理由: `credential-ledger` は utility 起点で screening 境界を限定しやすく、`saigai` は founder-side 例外確認が先行するため別レーン維持が妥当だから
- 少数意見: なし
- 再審条件: `credential-ledger` の opening が即時に branch 開放を要し、通常運転の parallelizable items を切り出せない場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`, `founder-office`
- 創業者拒否権: なし
