# One-Hundred-Fifty-Ninth Ordinary Sitting Ruling

## 裁定

- 裁定: `credential-ledger 高リスク審査着手議案` を discussion-1 で受理し、opening 範囲を `seat / auth boundary / encryption boundary / secret boundary` に限定したまま discussion-2 へ送る
- 理由: high-risk でも bounded opening に限定すれば、本流を止めずに screening 準備を進められるため
- 少数意見: なし
- 再審条件: opening 段階で新しい鍵や秘密情報の実利用が必要と判明した場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`
- 創業者拒否権: なし
