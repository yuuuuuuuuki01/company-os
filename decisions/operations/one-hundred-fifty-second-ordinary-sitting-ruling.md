# One-Hundred-Fifty-Second Ordinary Sitting Ruling

## 裁定

- 裁定: `常時稼働運用議案` を採択し、push 後 handoff 必須、active / pending の両棚維持、停止条件限定を通常運転の標準動作にする
- 理由: 24時間常時稼働に近づくには、push を停止点にせず、active PJ と pending review の両方を切らさない運転基準が必要なため
- 少数意見: なし
- 再審条件: post-push handoff の空欄が連続し、pending review 補充が止まる場合
- 実行責任者: `assembly-secretariat`, `project-management-office`, `portfolio-strategy`, `security-risk-compliance`
- 創業者拒否権: なし
