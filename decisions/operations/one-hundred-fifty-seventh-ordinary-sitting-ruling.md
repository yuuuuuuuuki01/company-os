# One-Hundred-Fifty-Seventh Ordinary Sitting Ruling

## 裁定

- 裁定: `高リスク審査着手順議案` を discussion-2 で維持し、最初の high-risk screening 候補を `credential-ledger`、policy-first 別レーンを `saigai` とする構成で deliberation に送る
- 理由: `credential-ledger` は high-risk utility として境界が比較的明確であり、`saigai` は policy-first として別扱いする方が gate 混雑を避けられるため
- 少数意見: なし
- 再審条件: `credential-ledger` と `saigai` の両方で同時に branch 開放が必要になった場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `founder-office`
- 創業者拒否権: なし
