# One-Hundred-Fifty-Sixth Ordinary Sitting Ruling

## 裁定

- 裁定: `高リスク審査着手順議案` を discussion-1 で受理し、最初の high-risk screening 候補を `credential-ledger` に固定したまま discussion-2 へ送る
- 理由: `credential-ledger` は utility 起点で境界が比較的明確であり、policy-first の `saigai` や追加の release-review capacity を要する項目より先に扱いやすいため
- 少数意見: なし
- 再審条件: `credential-ledger` で新しい秘密情報や鍵の使用が即時必要と判明した場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`
- 創業者拒否権: なし
