# One-Hundred-Forty-Sixth Ordinary Sitting Ruling

## 裁定

- 裁定: `事業並行改善運用議案` を採択し、active 事業は triad で作業を進め、inactive 事業は portfolio 台帳で観測しながら必要 seat と packet を整える
- 理由: active だけを見ると文脈が偏るため、portfolio 全体を見ながら必要な事業だけを段階的に前進させる必要がある
- 少数意見: なし
- 再審条件: portfolio 上の pending 事業が過剰に増え、triad の負荷が崩れる場合
- 実行責任者: `project-management-office`, `directorate`, `portfolio-strategy`, `people-talent`
- 創業者拒否権: なし
