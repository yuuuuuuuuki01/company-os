# One-Hundred-Sixtieth Ordinary Sitting Ruling

## 裁定

- 裁定: `credential-ledger 高リスク審査着手議案` を deliberation で採択し、bounded opening を formal review opening として開始する
- 理由: opening 範囲を `seat / auth boundary / encryption boundary / secret boundary` に限定し、branch 条件も先に切り分けているため、本流を止めずに high-risk screening を開始できるため
- 少数意見: なし
- 再審条件: opening 途中で新しい鍵または秘密情報の実利用が必要になり、approval branch が必要になった場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`
- 創業者拒否権: なし
