# One-Hundred-Sixty-First Ordinary Sitting Ruling

## 裁定

- 裁定: `Poin-T 高リスク審査準備議案` を discussion-1 で受理し、`credential-ledger` と同時 opening しない前提を維持したまま discussion-2 へ送る
- 理由: 次候補を ready 化しても、同時 opening しなければ gate capacity を守りつつ常時稼働を維持できるため
- 少数意見: なし
- 再審条件: `credential-ledger` の opening が想定より軽く、同時並行の再評価が必要になった場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`
- 創業者拒否権: なし
