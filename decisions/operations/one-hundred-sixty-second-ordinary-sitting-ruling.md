# One-Hundred-Sixty-Second Ordinary Sitting Ruling

## 裁定

- 裁定: `Poin-T 高リスク審査準備議案` を discussion-2 で維持し、`credential-ledger` と同時 opening しない前提を保ったまま deliberation に送る
- 理由: Poin-T を ready 化までに留めることで、high-risk gate の過負荷を避けながら次手を切らさずに済むため
- 少数意見: なし
- 再審条件: `credential-ledger` の opening が早期に閉じ、同時並行の再評価が必要になった場合
- 実行責任者: `security-risk-compliance`, `assembly-secretariat`, `release-review-committee`
- 創業者拒否権: なし
