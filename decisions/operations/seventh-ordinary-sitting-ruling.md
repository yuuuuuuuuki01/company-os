# Seventh Ordinary Sitting Ruling

## 短報

- 裁定: 第七回定例会は6件の報告を受理し、2件を `discussion-1` 完了として第八回へ送った。
- 理由: ロールアウト成果物が確認でき、残件2件も次段階に進める下書き資産がそろった。
- 少数意見: なし
- 再審条件: 通常動議または信任見直し
- 実行責任: `assembly-secretariat`, `people-talent`, `data-knowledge`
- 創業者拒否権: なし

## 報告受理

- `record-normalization-motion`: 優先記録の正規化完了を確認し、bootstrap 補修案件としては閉じる。
- `unit-state-separation-motion`: `units/registry.md` と `parallel-operations-board.md` に分離モデルが載ったことを確認し、以後は通常改定線で扱う。
- `unit-activation-gate-motion`: ルールと SOP の発効を確認し、次は最初の live activation review を監視する。
- `committee-execution-bridge-motion`: 委員会 findings の handoff 導線を監視対象へ移す。
- `audit-fast-path-motion`: `watch / route / urgent` の導線を稼働中として扱う。
- `ai-bulletin-intake-motion`: intake board 稼働と trial item の初回振分けを確認する。

## Discussion-1 結果

- `department-steward-seating-wave-motion`: wave board を作成し、wave-1 を高負荷ガバナンス部署に絞る方針で `discussion-2` へ送る。
- `decision-template-normalization-motion`: 短い日本語裁定テンプレート案を作成し、適用境界を詰めるため `discussion-2` へ送る。

## 次の引継ぎ

1. 第八回定例会を開く。
2. `people-talent` は wave-1 の steward 確認案を整える。
3. `data-knowledge` は短報裁定テンプレートの適用境界を整理する。
