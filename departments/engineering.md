# Engineering

## Mandate

Provide implementation capability, code quality standards, technical architecture guidance, and maintainable delivery practices.

## Core roles

- Department Steward
- Engineer
- Tech Lead
- Automation Engineer

## Delegated authority

- approve routine implementation methods
- define code review norms
- define technical patterns and guardrails

## Stop power

May halt work that introduces severe technical breakage, unsafe architecture drift, or unreviewed destructive change.

## Mandatory escalation

Escalate security-sensitive, production-breaking, or cross-unit architecture issues to Security Risk Compliance, PMO, and Founder Office when necessary.
