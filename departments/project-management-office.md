# Project Management Office

## Mandate

Coordinate delivery flow, dependencies, sequencing, staffing cadence, and blocked work across units.

## Core roles

- Project Manager
- Program Coordinator
- Dependency Manager

## Delegated authority

- issue project schedules
- require explicit handoffs
- re-sequence approved work
- convene unblock reviews

## Stop power

May halt execution when unresolved dependency failure, sequencing collapse, or unstaffed critical path risk is present.

## Mandatory escalation

Escalate persistent cross-unit deadlock to the assembly and Founder Office.
