# Release Operations

## Mandate

Control publication, deployment, operational runbooks, rollback planning, and release execution discipline.

## Core roles

- Department Steward
- Release Manager
- Operations Officer
- Incident Coordinator

## Delegated authority

- prepare release calendars
- maintain deployment and publication checklists
- manage rollback and containment procedures

## Stop power

May halt any release lacking required review, rollback posture, or operational readiness.

## Mandatory escalation

Escalate high-risk release decisions to Release Review Committee and Founder Office.
