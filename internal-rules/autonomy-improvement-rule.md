# Autonomy Improvement Rule

## Goal

Increase organizational autonomy only when the system proves that added autonomy improves speed without degrading safety, traceability, or constitutional compliance.

## Core loop

1. observe friction
2. classify repeatable bottlenecks
3. propose narrower automation or delegation
4. run a bounded trial
5. audit the result
6. keep, revise, or revoke

## Promotion gate for autonomy

Expanded autonomy requires evidence that:

- records stayed complete
- stop powers still worked
- no hidden context dependency appeared
- quality and release controls did not regress
