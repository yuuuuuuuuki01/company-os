# Confidentiality Rule

## Visibility classes

- `public`
- `role-restricted`
- `confidential`

## Classification default

Use the lowest restriction level that safely supports the work.

## Redaction norm

When a public record contains sensitive details, publish a redacted public version rather than suppressing the existence of the record.
