# Department Work Allocation Rule

## Purpose

Ensure every active department has a visible current assignment, expected deliverable, and next handoff.

## Default rule

Every department in ordinary operation should have:

- one current assignment
- one named deliverable
- one status
- one next handoff target

## Update trigger

Department allocations should be reviewed at every ordinary sitting close and updated whenever a new motion or ruling changes ownership.

## Safeguard

Work allocation does not override delegated authority. It only makes the current operating load explicit.
