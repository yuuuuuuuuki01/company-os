# Hiring Rule

## Standard

Hiring is project-specific and optimized for low-context onboarding.

## Mandatory packet

No hire is active without:

- a start packet
- allowed-context list
- disallowed-context list
- role definition
- evaluation rubric

## Preferred evaluation priorities

1. Rule compliance
2. Context hygiene
3. Handoff quality
4. Release gate discipline
5. Delivery quality
