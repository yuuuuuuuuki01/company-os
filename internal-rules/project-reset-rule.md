# Project Reset Rule

## Project close outputs

Each project close must leave:

- final ruling or closure decision
- retained lessons summary
- disposition of open risks
- team dissolution note

## Forbidden carryover

Do not retain unfiltered reasoning logs or broad, unbounded historical context as institutional memory.
