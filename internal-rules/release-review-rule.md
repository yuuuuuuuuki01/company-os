# Release Review Rule

## Required checks

- scope is documented
- risks are classified
- required reviewers are named
- rollback or containment plan exists when relevant
- founder sign-off is requested for high-risk actions

## Blocking authority

The Release Review Committee may block a high-risk act until the committee approves a release record.
