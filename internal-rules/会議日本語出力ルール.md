# 会議日本語出力ルール

## 目的

ordinary sitting と議案の本文を UI 上で読みやすく保つため、会議本文の標準出力言語を日本語に統一する。

## 適用範囲

- ordinary sitting の本文
- 議案本文
- 裁定本文
- 議長による質問、意見、改善案の促し

## ルール

- 本文の見出しは原則日本語を使う
- `Goal`、`Agenda`、`Definition of done` のような定型見出しは `目的`、`議題`、`完了条件` に置き換える
- UI で表示される議論本文、要点、理由は原則日本語で記録する
- frontmatter の機械項目は英語のままでよい

## 境界

- 既存の全過去文書を一括改名・一括翻訳する義務までは含めない
- ただし現在進行中または今後更新する会議文書は日本語優先で更新する
