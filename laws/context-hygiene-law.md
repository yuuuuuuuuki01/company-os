# Context Hygiene Law

## Low-context operating norm

Newly hired agents start with only the approved start packet and whitelisted materials.

## Whitelist rule

No actor may consult materials outside the allowed document list without explicit authorization recorded in the petition, ruling, or start packet.

## Project dissolution rule

At project close, detailed working memory and informal reasoning logs are not retained as institutional memory.

## Durable memory rule

Durable memory consists of:

- the constitutional stack
- laws, articles, and active rules
- rulings
- reusable summary knowledge
