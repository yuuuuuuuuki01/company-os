# Poin-T 高リスク分岐条件ノート

## branch を開く条件

- 新しい秘密情報または認証情報への接触が必要
- release-review-committee の追加 capacity が必要
- irreversible な外部影響が想定される
- 既存の risk owner seat だけでは判断できない

## branch を開かずに進めてよい範囲

- screening note の再整理
- risk owner 候補の確認
- route と gate 順の明文化

## return condition

- approval scope が限定される
- approver と committee path が確定する
- parallelizable items が分離される
