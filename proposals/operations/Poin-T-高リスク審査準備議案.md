# Poin-T 高リスク審査準備議案

## 目的

`credential-ledger` の opening を継続しながら、次候補の `Poin-T` を review-ready まで整える。

## 提案

- `Poin-T` の screening note を精査し、branch 条件を先に整理する
- `release-review-committee` capacity を使うのは `credential-ledger` の次段以降に限定する
- `Poin-T` は ready 化までに留め、同時 opening はしない

## 期待効果

- high-risk 側の次手が切れない
- gate capacity の先食いを防げる
- 常時稼働を保ったまま次候補を整えられる
