# 定型業務スクリプト化導入議案

## 目的

定型化した業務を自動的にスクリプトやCLIへ置換する原則を正式導入する。

## 提案

- 反復業務を `3回以上` で候補化する
- 低リスクなら script / cli へ置換する
- 全ての script を監査台帳へ登録する
- 監査機関が適正実行と漏れを継続点検する

## 期待効果

- トークン消費を抑えられる
- 反復処理を高速化できる
- スクリプトの形骸化を監査で抑制できる
