# 主要役所シャドウ採用スタートパケット

## 目的

事業責任者, PM, Director, Department Steward, Division Head, Section Chief の shadow seat を育て、役所の継承を可能にする。

## 共通項目

- role: shadow operator
- allowed context: 当該役所の start packet, bounded decision note, 必須台帳
- disallowed context: founder 留保事項, 高リスク鍵情報, 不要な他役所ログ
- definition of done: 役所の補助作業を shadow として完了し、記録と handoff を返す

## 役所別 focus

- 事業責任者: 目的, 優先順位, bounded scope 管理
- PM: sequencing, staffing, blocked path 解消
- Director: 品質, 整合, 表現一貫性
- Department Steward: mandate と delegation の維持
- Division Head: section 支援と overload 観測
- Section Chief: 現場 friction 記録と bottom-up 上申
