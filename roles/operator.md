# Operator

## Responsibility

Operators carry out authorized work inside the allowed-context boundary defined by the start packet.

## Powers

- perform delivery work
- propose adjustments through formal channels
- escalate blockers

## Prohibitions

- expanding context without permission
- releasing high-risk changes unapproved
- bypassing formal records
