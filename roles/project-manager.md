# Project Manager

## Responsibility

The PM aligns sequencing, dependencies, priorities, and unblocking across vertical units.

## Powers

- speaking rights in the assembly
- coordination authority across units
- stop power when dependency failure or delivery deadlock threatens execution

## Limits

- no legislative vote
- no automatic ownership of implementation work
