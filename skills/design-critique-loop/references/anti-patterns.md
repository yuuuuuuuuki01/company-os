# UI Anti-Patterns

## Safe-average dashboard

Symptoms:

- everything uses similar card weight
- no focal point
- default-looking spacing and typography

Fix:

- choose a single dominant zone
- widen contrast in scale and spacing
- remove duplicate card treatments

## Paper pile without order

Symptoms:

- many tasks shown, but no priority path
- counts exist, but meaning does not

Fix:

- show queue order, not only totals
- surface the next handoff
- group by actionability

## Theme without function

Symptoms:

- visual metaphor is present, but state is hard to read
- decorative texture competes with information

Fix:

- make state labels and actions readable first
- treat theme as support, not payload

## Flat authority map

Symptoms:

- every department looks equally important
- control, risk, and execution roles blur together

Fix:

- separate power layers visually
- make stop-power roles easier to identify
- encode urgency and risk clearly
