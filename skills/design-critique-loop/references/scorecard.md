# Design Scorecard

Score each item from `1` to `5`.

## Clarity

- Can a new viewer tell what this screen is for?
- Is the main action obvious within a few seconds?

## Hierarchy

- Is there a clear order of attention?
- Do headings, actions, and support details separate cleanly?

## Density

- Is the screen overpacked or under-expressive?
- Do related elements group together naturally?

## Contrast

- Do type, color, spacing, and shape create enough distinction?
- Are important actions visibly stronger than passive content?

## Motion

- Would motion clarify transitions, or would it only decorate?
- Is movement tied to state change and focus?

## Product Fit

- Does the visual language match the product's role and audience?
- Does it feel intentional rather than generic?

## Summary format

Use this compressed format:

`clarity 2 / hierarchy 2 / density 3 / contrast 2 / motion 1 / fit 2`
