# SOP: Issue Committee Finding Handoff

1. Name the finding type and whether it is binding or advisory.
2. Name one executor.
3. Name one deadline or next sitting checkpoint.
4. Name the receiving lane or owner.
5. Record the handoff in the relevant board.
6. Escalate missed handoff acceptance to PMO or the chair.
