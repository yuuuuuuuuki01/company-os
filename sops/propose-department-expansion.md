# SOP: Propose Department Expansion

1. Define the growth need in short form: what is missing, overloaded, or structurally blocked.
2. Classify the request as create, split, merge, mandate-change, officer-line, or stop-power change.
3. Name the sponsoring `Department Steward` and one seconder.
4. File the request through a formal petition.
5. Add the request to the department growth board.
6. Request compatibility review from Constitutional Affairs.
7. Request impact review from any affected departments.
8. Route the motion to the lawful approval body.
9. Record approve, reject, pause, or revise in the board.
10. Update the department registry and delegation matrix only after approval becomes effective.
