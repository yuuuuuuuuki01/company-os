# SOP: Request Key Access

1. Confirm the task cannot proceed without the key.
2. Assign or identify the `Key ID`.
3. File the request with purpose, scope, and minimum duration.
4. Record checkout in the key loan ledger before handing over the key.
5. Record return, revocation, or compromise immediately after the access window ends.
