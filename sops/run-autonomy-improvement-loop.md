# SOP: Run Autonomy Improvement Loop

1. Collect weekly friction from audit reports, AI bulletins, and operating records.
2. Select one narrow autonomy candidate.
3. Define success and rollback conditions.
4. Run a bounded trial.
5. Audit the trial in short Japanese form.
6. Convert the result into a keep, revise, or revoke petition.
