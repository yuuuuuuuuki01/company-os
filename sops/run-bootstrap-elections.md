# SOP: Run Bootstrap Elections

1. Open the inaugural assembly packet.
2. Elect the assembly chair and floor clerk.
3. Run Unit Owner elections for units entering active operation.
4. Elect the three standing committees.
5. Elect the Constitutional Guardian.
6. Elect PM and Director if cross-unit work is expected.
7. Write every outcome to the officeholder registry.
