# SOP: Run Company OS Improvement Sitting

1. Open with the latest audit note, AI bulletin, current activity board, and improvement board.
2. Batch `3` to `7` agenda items using urgent / next / later priority.
3. Label each item as `report`, `discussion`, or `deliberation`.
4. For substantive items, move them through `discussion-1`, `discussion-2`, and `deliberation`.
5. Route each active item to a short motion, referral, bounded trial, or report acknowledgment.
6. Assign owner, reviewer, next handoff, and next sitting before closing.
7. Open or update the next ordinary sitting record as the closing trigger.
8. Record what was activated, deferred, reported, adopted, or closed.
9. Keep the sitting output short and in Japanese.
