# SOP: Run Department Work Allocation

1. Review active motions, rulings, and improvement boards.
2. Assign one current priority to each active department.
3. Name one deliverable and one next handoff for each department.
4. Mark departments with no immediate load as `ready`, not `active`.
5. Update the department work allocation board before closing the sitting.
