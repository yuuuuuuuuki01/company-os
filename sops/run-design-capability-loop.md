# SOP: Run Design Capability Loop

1. Collect one weak UI artifact or one unclear product surface.
2. Run a short diagnosis using the design scorecard.
3. Name the strongest failure mode.
4. Select one drill from the design skillset.
5. Produce one improved direction, not multiple half-finished variants.
6. Review the result with Product and Service Design and Directorate.
7. Record one reusable lesson in the design capability backlog.
