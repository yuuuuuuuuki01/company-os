# SOP: Run Operating Audit

1. Select the audit period and target scope.
2. Inspect rule stack changes, releases, and exception records.
3. Test a sample of motions, reviews, and release approvals.
4. Classify findings by severity and recurrence.
5. Publish the audit opinion and remediation recommendations.
6. Route material irregularity to Founder Office and Constitutional Affairs.
