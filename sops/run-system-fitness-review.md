# SOP: Run System Fitness Review

1. Treat system fitness review as the first organizational mission after bootstrap.
2. Review whether the Company OS is internally coherent, enforceable, auditable, and proportionate.
3. Test rule hierarchy, delegation boundaries, reporting burden, and stop-power collisions.
4. File a short Japanese report with opinion, top risks, and immediate fixes.
5. Route any structural fix through a formal petition.
