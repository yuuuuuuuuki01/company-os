# スクリプト監査ループ

## 手順

1. スクリプト台帳から `active` と `recently changed` を拾う
2. 適用範囲、例外条件、branch 条件、rollback 手段を確認する
3. 台帳と実行結果の整合を点検する
4. `適正 / 条件付き適正 / 是正必要 / 停止勧告` を記録する
5. 必要なら founder-office と security-risk-compliance へエスカレーションする

## 出力

- 監査短報
- 是正要求
- 停止勧告
