# 全部署論点総点検ループ

## 手順

1. assembly-secretariat が総点検テーマ、提出期限、優先観点を宣言する
2. 各部署は [全部署論点総点検台帳.md](C:/agent/codex/company-os/ledgers/全部署論点総点検台帳.md) に提出行を追加する

4. assembly-secretariat が整理済み論点を `一括審議束` にまとめる
5. ordinary sitting で `report -> discussion-1 -> discussion-2 -> deliberation` の順に進める
6. 採択後は各部署へ返送し、watch または実装へ落とす
