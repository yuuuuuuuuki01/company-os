# 定型業務スクリプト化ループ

## 手順

1. 反復している処理を `定型候補` として台帳に記録する
2. `script`, `cli`, `template` のどれで置換すべきかを決める
3. 低リスクならスクリプトを実装し、適用範囲と禁止範囲を明記する
4. 台帳と監査対象を更新する
5. 監査機関へ `スクリプト監査対象` として送る

## 出力

- script または cli entry
- 適用範囲ノート
- 監査対象台帳更新
