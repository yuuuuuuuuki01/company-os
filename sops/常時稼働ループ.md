# 常時稼働ループ

## 手順

1. active PJ の bounded work を確認する
2. pending business の review-ready / review-opened item を確認する
3. push 後は `next handoff` が空欄でないことを確認する
4. empty になった棚は `project-management-office` と `portfolio-strategy` が補充する
5. high-risk と branch 以外は次作業へそのまま進める

## 出力

- current activity 更新
- portfolio queue 更新
- ordinary sitting または review opening の次手
