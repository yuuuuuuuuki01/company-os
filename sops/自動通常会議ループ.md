# 自動通常会議ループ

## 手順

1. `company-os-improvement-board` から active item を 3 から 7 件束ねる
2. report item を先頭に置き、discussion と deliberation を後ろに置く
3. 各 substantive item で質問、意見、建設的改善案を確認する
4. reusable artifact を最低1つ残す
5. ruling を短く記録する
6. 次回 ordinary sitting を開く
7. `current-activity-board` と `department-work-allocation-board` を更新する
8. `git add`, `git commit`, `git push origin main` で反映する

## 停止条件

- Founder 返却条件に触れる
- blocking objection が残る
- reusable artifact が作れない
