# 非稼働事業審査起票ループ

## 手順

1. [非稼働事業着席準備台帳.md](C:/agent/codex/company-os-sync/ledgers/非稼働事業着席準備台帳.md) で `bundle-issued` 以上の事業を拾う
2. [非稼働事業審査順序台帳.md](C:/agent/codex/company-os-sync/ledgers/非稼働事業審査順序台帳.md) の queue 順に `review opening` を作る
3. low / medium は ordinary sitting の議案へ束ねる
4. high-risk / policy-first は gate owner が review opening を出し、必要なら approval branch を開く
5. 裁定後に `decision record` を台帳へ記録する

## 出力

- review opening proposal
- decision record link
- branch link if escalated
